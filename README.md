Get Started :
-------------

  Install Sails & Bower globally :
  ```
  sudo npm -g install sails
  npm install -g bower
  ```
  
  Install Dependencies for Sails & Bower :
  ```
  npm install
  bower install
  ```
  Launch Sails server : `sails lift` or with the debug mode : `sails debug`
