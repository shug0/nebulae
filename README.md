Get Started :
-------------
Sails Installs :

  install Sails & Bower in global
  ```
  sudo npm -g install sails
  npm install -g bower
  ```
  
  install Dependencies for Sails & Bower
  ```
  npm install
  bower install
  ```
  Launch Sails server
  `sails lift`
